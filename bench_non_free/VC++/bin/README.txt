In this folder you can find the programs compiled with VC++ 2015, with 64 bits and optimized.

If don't run in your computer , probably , need to install the redistributable 2015 package.

The run_short_benchmark.bat create a file of 800 M with random information , and after delete it.

The run_benchmark_numbers.bat, run_benchmark_strings.bat, and run_benchmark_objects.bat create a file of 1000 M
with random information, and when finish, delete it
